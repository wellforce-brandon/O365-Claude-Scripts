## Customization Guide

### For Different Stacks

**Python/FastAPI**:
- Update `backend-dev-guidelines.md`
- Change build checker to use `mypy`
- Update file patterns to `*.py`
- Use `black` instead of `prettier`

**Go**:
- Use `go-desktop-agent-guidelines.md` as base
- Change build to `go build`
- Update patterns to `*.go`
- Use `gofmt` for formatting

**Ruby/Rails**:
- Create `rails-dev-guidelines.md`
- Change build to `bundle exec rake`
- Update patterns to `*.rb`
- Use `rubocop` for linting

**PHP/Laravel**:
- Create `laravel-dev-guidelines.md`
- Change build to `composer validate`
- Update patterns to `*.php`
- Use `php-cs-fixer` for formatting
